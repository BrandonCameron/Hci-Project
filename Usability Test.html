Usability Test

For the usability test i asked the participants to complete 3 tasks.

Task 1 : Search for videos on a specified topic

Task 2 : Share a video

Task 3 : Upload a video

Participant 1 :

Task 1 :

I specifically asked for the user to search for power tools
no problems arose

clicked on video to share rather then use the share button from
the home page

Task 3 :

to be clicked into

The Search function and upload function seem to be understanble to users unfamilar
with the website. I attribute the time to find the upload function to lack of experince

share function present on the home page. This is most likely due to the function being behind
a three dot icon that only appears when hovering over a video
